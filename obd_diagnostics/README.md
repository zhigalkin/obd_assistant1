# 🚗 OBD-II Диагностика

Профессиональное локальное приложение для диагностики автомобиля через OBD-II адаптер с AI-анализом.

## Возможности

- 📡 **Живые данные** — все доступные датчики в реальном времени
- 🔍 **Полное сканирование** — все PIDs + расширенные протоколы для Audi/VW и BMW
- ⚠️ **Умная детекция аномалий** — даже без кодов ошибок ECU
- 🤖 **AI анализ** — gpt-4o-mini расшифровывает и даёт рекомендации
- 📚 **История сессий** — SQLite база данных
- 📊 **Дашборд** — Streamlit веб-интерфейс с графиками
- 💾 **Экспорт** — отчёты в .txt

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
# Главное приложение
python main.py

# Дашборд (в отдельном терминале)
streamlit run dashboard.py
```

## Настройка

1. Укажи **COM-порт** адаптера в config.py (или в интерфейсе)
2. Укажи **OpenAI API Key** в config.py или переменной окружения `OPENAI_API_KEY`
3. Выбери **марку автомобиля** (audi_vag / bmw / generic)

## Поддерживаемые протоколы

- **Стандарт:** SAE J1979 (все OBD-II PIDs)
- **Audi/VW (VAG):** давление наддува, лямбда, VCDS-совместимые PIDs
- **BMW:** специфические PIDs масла, DSC, iDrive

## Адаптеры

Рекомендуется: ELM327 v1.5 (USB или Bluetooth), OBDLink MX+
