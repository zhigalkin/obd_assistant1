📦 КАК СОБРАТЬ .EXE ДЛЯ WINDOWS

1. Скопируй всю папку obd_diagnostics на Windows компьютер

2. Открой cmd или PowerShell в папке obd_diagnostics

3. Запусти build.bat:
   > build.bat

4. Дождись окончания сборки (2-5 минут)

5. Готовый .exe будет в папке dist\OBD_Diagnostics.exe

6. Скопируй OBD_Diagnostics.exe куда хочешь и запускай!

─────────────────────────────────────────────
⚙️ ВАЖНО ДЛЯ ПЕРВОГО ЗАПУСКА:

1. Укажи API ключ OpenAI в настройках программы
2. Выбери правильный COM-порт твоего OBD адаптера
3. Выбери марку машины (generic / audi_vag / bmw)

─────────────────────────────────────────────
⚠️ ЕСЛИ ВОЗНИКЛИ ПРОБЛЕМЫ:

- Нет Python? Скачай: https://www.python.org/downloads/
- Антивирус блокирует? Добавь в исключения
- Ошибка "module not found"? Запусти: pip install -r requirements.txt
