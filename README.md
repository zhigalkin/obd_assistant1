# Прототип интеграции python-OBD с GPT-3.5-turbo

Это базовый прототип, который:

- Подключается к OBD-II адаптеру через python-OBD
- Считывает обороты, скорость и температуру охлаждающей жидкости
- Отправляет данные в OpenAI GPT-3.5-turbo для анализа
- Выводит рекомендации и возможные неисправности

## Требования

- Python 3.8+
- Установлены библиотеки: `obd`, `openai`
- Переменная окружения `OPENAI_API_KEY` установлена с вашим ключом OpenAI

## Запуск

```bash
export OPENAI_API_KEY="ваш_ключ"
python python_obd_gpt35_integration.py
```

Или в Windows PowerShell:

```powershell
$env:OPENAI_API_KEY="ваш_ключ"
python python_obd_gpt35_integration.py
```

## Примечания

- Пока что прототип выводит данные и диагностику в консоль
- В следующим этапах можно добавить интеграцию с телеграм-ботом и live-режим
